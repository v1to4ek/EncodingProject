# Binary Signal Encoder

Учебное консольное приложение на C#, демонстрирующее работу различных методов линейного кодирования цифровых данных.
Программа преобразует бинарную последовательность в выбранный формат кодирования и строит изображение полученного сигнала. 
Проект предназначен для изучения принципов представления и передачи цифровой информации.
